# 🐧 Linux Backup Script

Projeto simples desenvolvido em Linux/WSL utilizando Bash Script para automatizar backup de arquivos.

## 🚀 Funcionalidades

- Criação automática de pasta de backup
- Compactação de arquivos com tar
- Organização de diretórios
- Automação simples no terminal Linux

## 🛠 Tecnologias utilizadas

- Linux
- WSL
- Bash Script

## 📁 Estrutura do projeto

```bash
backup_projeto/
├── arquivos/
├── backups/
├── backup.sh
└── README.md
```

## ▶️ Como executar

Dar permissão ao script:

```bash
chmod +x backup.sh
```

Executar:

```bash
./backup.sh
```

- automação utilizando Bash Script.
